== Description ==

TW Sidebar Slider offers you to show Twitter some assistance with feeding straightforwardly onto your site. This sidebar plugin is a premium quality Wordpress plugin that accompanies an expansive scope of components with the goal that you can modify your TW Sidebar tweets to coordinate your site's topic and configuration.
Moreover, TW Sidebar Slider plugin is a simple yet effective Wordpress plugin that can be add a sidebar to your page at the left/right of the page.

= Features of TW Sidebar Slider: =

Very easy to install, setup and use.
It will be install in seconds.
Lots of customize options
Customizable link and border Colors
Change Alignment position, Left/Right

== Installation ==

How to Use:
Download "twitter_sidebar_slider.zip" to your computer.
Login wordpress as administrator
Click on Plugins --> Add New
On "Install Plugins" page, click on "Upload"
Browse and select " twitter_sidebar_slider.zip" and click on "Install Now"
You can activate it now.
Go to your "Tw Sidebar Slider" Menu and adjust the settings
